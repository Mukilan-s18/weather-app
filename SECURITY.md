# Security Policy

## Supported Versions

Only the latest release of AeroWeather is supported for security updates. If a vulnerability is found, we recommend upgrading to the latest commit on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |
| < v0.x  | :x:                |

## Reporting a Vulnerability

We take the security of our application seriously. If you find a security vulnerability, please do **not** open a public GitHub issue. Instead, follow these steps:

1. Draft a detailed description of the vulnerability, including:
   - Steps to reproduce (proof of concept).
   - The potential impact.
   - Any possible mitigations.
2. Email your report to the project owner/maintainer.
3. We will acknowledge your report within 48 hours and work with you to resolve the issue in a timely manner.
